# flower_classifier

This is a flower classifier built with fastai library. What kind of flowers can you classify?

The following flower types are supported:

- Rose
- Gerbera
- Tulip
- Lily
- Orchid
- Iris
- Carnation
- Sunflower
- Hydrangea
- Marigold
- Lotus
- Dahlia
- Gladioli
- Chrysanthemums
- Camellia
- Peony
- Lavender
- Snapdragon
- Hibiscus
